# Medical Diagnostics - Система записи на медицинскую диагностику

Django-приложение для управления записями на медицинскую диагностику.

## Описание

Веб-приложение для медицинского центра, которое позволяет пациентам:
- Просматривать доступные услуги
- Записываться на прием к врачам
- Просматривать результаты диагностики
- Управлять своими записями

## Технологии

- **Django 4.2.7** - веб-фреймворк
- **Bootstrap 5** - CSS-фреймворк
- **CKEditor** - текстовый редактор
- **Crispy Forms** - стилизация форм
- **Pillow** - работа с изображениями
- **SQLite** - база данных (для разработки)

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/doboleshev/medical_diagnostics.git
cd medical_diagnostics
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
```

3. Активируйте виртуальное окружение:
- Windows:
```bash
venv\Scripts\activate
```
- Linux/Mac:
```bash
source venv/bin/activate
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

5. Примените миграции:
```bash
python manage.py migrate
```

6. Создайте суперпользователя:
```bash
python manage.py createsuperuser
```

7. Запустите сервер разработки:
```bash
python manage.py runserver
```

8. Откройте браузер и перейдите на `http://127.0.0.1:8000/`
